# 👥 HR Analytics | Employee Attrition Analysis

## 📌 Project Overview
The HR Analytics project focuses on analyzing employee attrition trends and workforce patterns to understand the key factors influencing employee turnover. Using data analytics and visualization techniques, this project provides actionable insights into employee demographics, departmental attrition, work-life balance, promotions, income, and overall workforce performance.

The project demonstrates end-to-end analytics through data cleaning, KPI generation, and dashboard development.

---

## 🎯 Project Objectives
- Analyze employee attrition trends
- Identify factors affecting employee turnover
- Compare attrition across departments and job roles
- Study work-life balance and promotion impact
- Understand salary and hourly rate patterns
- Create interactive dashboards for HR decision-making

---

## 📊 Key Performance Indicators (KPIs)

This project analyzes the following HR metrics:

- Total Employees
- Active Employees
- Male Attrition
- Female Attrition
- Attrition Rate
- Average Working Years by Department
- Attrition Rate by Department
- Attrition vs Years Since Last Promotion
- Job Role vs Work-Life Balance
- Attrition vs Monthly Income
- Average Hourly Rate Analysis
